# Suarez Dienstleistungen – Website

Sitio web estático preparado para GitHub Pages.

## Publicación
1. Copiar todos los archivos de esta carpeta a la raíz del repositorio.
2. Hacer commit y push a `main`.
3. En GitHub: Settings → Pages → Deploy from a branch → `main` → `/root` → Save.

## Importante antes de promocionarla ampliamente
Completar y enlazar correctamente el **Impressum** y la **Datenschutzerklärung** con los datos legales reales de la empresa.
